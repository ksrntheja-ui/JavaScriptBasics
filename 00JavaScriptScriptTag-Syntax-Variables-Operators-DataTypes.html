JavaScript Programs

01.Topic : JavaScript Statements [var -- Declares a variable]

JavaScript uses the var keyword to declare variables.

In HTML, JavaScript programs are executed by the web browser.

JavaScript statements are composed of : Values, Operators, Expressions, Keywords, and Comments.

02.Topic : JavaScript Syntax

  • The JavaScript syntax defines two types of values: Fixed values [literals] and variable values [variables : All JavaScript variables must be identified with unique names -> identifiers.]
  • JavaScript Identifiers: First character must be a letter, or (_), or ($). Numbers are not allowed as the first character. This way JavaScript can easily distinguish identifiers from numbers.
  • Is Case Sensitive
  • JavaScript does not interpret VAR or Var as the keyword var.
  • Single Line Comments - //. Multi-line Comments - /* */.

03.Topic : JavaScript Variables

  • Creating a variable in JavaScript is called "declaring" a variable.
    • Declare a JavaScript variable with the var keyword: var carName
    • After the declaration, the variable has no value : (undefined)
    • To assign a value to the variable, use : equal sign
    • If you re-declare a JavaScript variable, it will not lose its value.
